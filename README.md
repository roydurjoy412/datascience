Introduction to Data Science
Midterm Project
Project :
Download a valid dataset from a trusted online source and intentionally introduce a few data issues (such
as missing values, incorrect entries, inconsistent formats, duplicate rows etc). Then, apply appropriate
data preparation and cleaning steps to correct those issues. After that, perform exploratory data
analysis by calculating and interpreting descriptive statistics for the dataset. If students collect primary
data, they may use it for the project instead of the secondary dataset. In that case, they should perform all
necessary steps, including data cleaning, preparation, and exploratory analysis, before calculating and
interpreting descriptive statistics.
In case of using secondary data, students must include the link or proper reference to the dataset source
in their report.
Project Deliverables

execute and explain parts of your R code during the session.
